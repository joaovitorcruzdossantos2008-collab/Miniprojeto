# Miniprojeto

Projeto em Java que simula um sistema simples de contas bancárias utilizando Programação Orientada a Objetos (POO), ArrayList, laços de repetição e estruturas condicionais (if/else).
Funcionalidades:
Classe ContaBancaria com:
numeroConta
saldo
nomeTitular
Lista de contas bancárias com diferentes saldos
Percorrimento da lista com loop
Comparação de valores com if/else
Identificação da conta com o maior saldo
Conceitos aplicados:
Encapsulamento
Construtores
Métodos (toString)
Estruturas de repetição
Condicionais
Manipulação de listas
Lógica de programação
POO
